# GitHub Viewer

A sleek GitHub username viewer built with React + TypeScript + ShadCN UI + Tailwind CSS.

## 🔍 Features

- GitHub username input form
- List of repositories
- Commit activity chart (weekly & daily)
- Responsive design

## 🚀 Tech Stack

- React + TypeScript
- Tailwind CSS
- ShadCN UI
- Recharts
- GitHub REST API

## 🛠️ Getting Started

```bash
npm install
npm run dev
